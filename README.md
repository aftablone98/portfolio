## 📖 About

This portfolio website was designed and developed from scratch using **HTML, CSS, and JavaScript**. It serves as my online portfolio where I showcase my projects, technical skills, resume, and contact information.

The website features a clean, responsive design that works seamlessly across desktops, tablets, and mobile devices.
